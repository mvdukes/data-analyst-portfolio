# Revenue & Expense Dashboard (Dick's Sporting Goods)

**Tools:** Excel, SEC Financial Dataset

**Goal:** 
Build a dashboard showing revenue, expense, and profit over the trailing four quarters using real data from SEC filings.

**Steps:**
1. Pulled Dick’s Sporting Goods financial data from SEC `num.txt` and `sub.txt` files.
2. Filtered for tags like Revenue, Cost of Goods Sold, and Net Income.
3. Cleaned and formatted in Excel.
4. Built PivotTables and a Profit calculated field.
5. Designed a dashboard showing trends by quarter over the trailing four quarters.

**Skills Demonstrated:**
- Data cleaning and transformation (Excel)
- Financial analysis (Profit margin trends)
- Dashboard design (PivotCharts, slicers)

**Insights:**
- Consistent quarterly revenue growth.
- Expense growth proportional to sales.
- Net profit margin over the trailing four quarters is 64%.
